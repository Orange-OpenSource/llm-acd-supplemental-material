# README — Supplemental Material

This repository contains the supplemental material for the paper "From Zero-Shot to Reward-Aware: Evaluating Prompting and Memory in LLM-Based Cyber Defense Agents".

## Repository structure

- `LLM Logs`  
  This folder contains the logs of the LLM agents.

- `Prompts`  
  This folder contains the system prompts (the various prompt variants) described in the paper.

- `RL Logs`  
  This folder contains the logs of the RL agents.

- `Scenario Configuration Files`  
  This folder contains the configuration files for Cyborg to recreate the network topologies used in the experiments.

## Notebooks (at repository root)

- `llm-agent.ipynb`  
  Notebook containing the code to replay the experiments for the LLM agents.

- `rl-agent.ipynb`  
  Notebook containing the code to replay the experiments for the RL agents.
