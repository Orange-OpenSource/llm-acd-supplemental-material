# Scenario Configuration Files

This folder contains the configuration files for the three network topology variants. Each variant requires a YAML file and a CFG file. The `llm-agent.ipynb` and `rl-agent.ipynb` notebook will require a path toward the folder corresponding to a toppology and which contains the YAML and CFG file.
